# faucet-testnet
Api for providing BDX testnet faucets


>PROCEDURE TO BUILD
```
 cd faucet-testnet 
 mkdir build
 cd build
 cmake ..
 make
 ./testnetfaucet
```
>FOR JSON

`After build the faucet if you need to print the json in termianl means just add`

`./testnetfaucet --json` | `./testnetfaucet -json`

>FOR USER RPC

`./testnetfaucet --ip-port="ip:port"`
